Ansys Mechanical is the host application for Mesh Workflows, providing the UI framework, geometry handling, and simulation environment.

## Role in Mesh Workflows

- Provides the Tree Outline, Details View, and Context Tab UI
- Manages geometry import and CAD connectivity
- Hosts the Control Panel, Domain Browser, and Property Worksheet
- Transfers geometry to PrimeMesh for meshing and receives results back
- Manages Named Selections, which become Labels in the workflow

## Access Methods

Mesh Workflows can be accessed via:
- Tree Outline right-click on Mesh
- Model Context menu
- Geometry Window context menu
