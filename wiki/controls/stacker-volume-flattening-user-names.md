Stacker: Volume Flattening User Names sets a custom name for the base face created during Volume Flattening.

## Overview

The **Stacker: Volume Flattening User Names** control provides a user-defined label for the base face created by the Stacker: Volume Flattening control.

## Key Parameters

| Parameter | Description |
|---|---|
| **Base Face Label** | Name for the created base face (publishable) |

## Steps That Use This Control

- Stacker workflow steps (used alongside Stacker: Volume Flattening)
