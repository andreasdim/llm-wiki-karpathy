Volumes Merging merges all volumes that share input faces defined in the scope into a single volume.

## Overview

The **Volumes Merging** control combines multiple volumes sharing common faces into one volume. Custom names can be assigned to the resulting volume zone and label.

## Key Parameters

| Parameter | Description |
|---|---|
| **Define By** | Standard scope options |
| **Scoping Method** | Standard scope options |
| **Volume Zone Name** | Custom name for the created volume zone |
| **Volume Label Name** | Custom name for the created volume label |

## Steps That Use This Control

- Volume consolidation steps
- Assembly mesh preparation
