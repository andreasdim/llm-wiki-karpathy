Topology Diagnostics User Names provides custom names for labels created during topology diagnostics.

## Overview

The **Topology Diagnostics User Names** control assigns user-defined names to the diagnostic labels created by the Topology Diagnostics control. If names are not specified, default names are used. The Topology Diagnostics outcome provides the diagnostic information.

## Key Parameters

| Parameter | Description |
|---|---|
| **Incomplete Boundaries Label Name** | Name for incomplete boundaries label (publishable) |
| **Missing Boundaries Label Name** | Name for missing boundaries label (publishable) |
| **Invalid Boundaries Label Name** | Name for invalid boundaries label (publishable) |
| **Intersecting Pairs Label Name** | Name for intersecting pairs label (publishable) |
| **Overlapping Faces Label Name** | Name for overlapping faces label (publishable) |
| **Thin Stripes Label Name** | Name for thin stripes label (publishable) |
| **Small Edges Label Name** | Name for small edges label (publishable) |
| **Unconnected Free Edges Label Name** | Name for unconnected free edges label (publishable) |

## Steps That Use This Control

- Topology diagnostic steps (used alongside Topology Diagnostics)
