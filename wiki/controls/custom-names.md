Custom Names assigns user-defined names to enclosures or extrusions.

## What It Does

Custom Names lets you provide specific names for labels, zones, and volumes created by enclosure or extrusion operations. All name fields are publishable to the Property Worksheet.

## Key Parameters

| Parameter | Description |
|---|---|
| **Internal Label Name** | Name for the label attached to internal entities |
| **External Label Name** | Name for the label attached to external entities |
| **Volume Zone Name** | Name for the volume zone |
| **Volume Label Name** | Name for the volume |
| **Face Zone Name** | Name for the face zone |

## Steps That Use This Control

Used alongside enclosure and extrusion controls to assign meaningful names to created entities.
