Solder Ball User Names specifies custom names for labels created during solder ball creation.

## Overview

The **Solder Ball User Names** control assigns user-defined names to the various solder ball region labels (center, top, bottom, pads, pad faces). All name fields are publishable to the Property Worksheet.

## Key Parameters

| Parameter | Description |
|---|---|
| **Center Solder Label Name** | Name for center solder label |
| **Top Solder Label Name** | Name for top solder label |
| **Bottom Solder Label Name** | Name for bottom solder label |
| **Top Solder Pad Label Name** | Name for top solder pad label |
| **Bottom Solder Pad Label Name** | Name for bottom solder pad label |
| **Top Solder Pad Face Label Name** | Name for top solder pad face label |
| **Bottom Solder Pad Face Label Name** | Name for bottom solder pad face label |

## Steps That Use This Control

- Solder ball meshing steps (used alongside Solder Ball Creation)
