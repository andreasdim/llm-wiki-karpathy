Topology Deletion defines the scope to delete topology from the mesh.

## Overview

The **Topology Deletion** control removes topology entities from scoped regions. It supports standard Define By, Scoping Method, and Scoping Pattern options.

## Key Parameters

| Parameter | Description |
|---|---|
| **Define By** | Value or Outcome |
| **Scoping Method** | Standard scope options |
| **Scoping Pattern** | Regex pattern for entity selection |

## Steps That Use This Control

- Topology cleanup steps
- Pre-meshing topology simplification
