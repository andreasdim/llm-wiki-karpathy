Stackable Bodies Detection User Names sets a custom name for detected stackable bodies.

## Overview

The **Stackable Bodies Detection User Names** control provides a user-defined name for the labels created by the Stackable Bodies Detection control.

## Key Parameters

| Parameter | Description |
|---|---|
| **Name** | Name for the stackable bodies (publishable) |

## Steps That Use This Control

- Stacker workflow steps (used alongside Stackable Bodies Detection)
