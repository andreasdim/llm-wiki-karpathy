Custom Names Sizing assigns a user-defined name to a created size field.

## What It Does

Custom Names Sizing lets you provide a specific name for a size field created during sizing operations. This is useful for identifying and referencing size fields later in the workflow.

## Key Parameters

| Parameter | Description |
|---|---|
| **Name** | Name for the created size field |

## Steps That Use This Control

Used alongside sizing controls to assign meaningful names to size fields.
