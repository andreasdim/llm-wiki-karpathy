Number of Divisions on Edges sets the number of element divisions along the stacking direction for scoped edges.

## Overview

The **Number of Divisions on Edges** control specifies how many divisions are created along the stacking direction. Scoping is by Label or Zone (not Part).

## Key Parameters

| Parameter | Description | Default |
|---|---|---|
| **Define By** | Value or Outcome | -- |
| **Scoping Method** | Label or Zone | -- |
| **Number Of Divisions** | Number of divisions along the stacking direction (parametrizable, publishable) | -- |

## Steps That Use This Control

- Stacker mesh workflow steps
- Sweep-based volume meshing steps
