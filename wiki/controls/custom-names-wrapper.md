Custom Names Wrapper assigns user-defined names to entities created by a wrapping operation.

## What It Does

Custom Names Wrapper lets you provide specific names for the part, volume zone, face zone, and edge labels created during a wrap operation.

## Key Parameters

| Parameter | Description |
|---|---|
| **Part Name** | Name for the part enclosing the volume |
| **Volume Zone Name** | Name for the volume zone of the created part |
| **Face Zone Name** | Name for the face zone forming the boundary |
| **Edge Label Name** | Label name for created feature edges |

## Steps That Use This Control

Used alongside wrapping controls such as [[constant-size-wrapper]] to name created entities.
