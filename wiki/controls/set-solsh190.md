Set SOLSH190 assigns the SOLSH190 element type to scoped entities.

## Overview

The **Set SOLSH190** control changes the element type of scoped entities to SOLSH190, a solid-shell element type used for thin-walled structures.

## Key Parameters

| Parameter | Description |
|---|---|
| **Define By** | Value or Outcome |
| **Scoping Method** | Standard scope options |
| **Scoping Pattern** | Regex pattern for entity selection |

- Element type assignment steps in structural workflows
