Outcome that allows scoping the created entities of an operation, with options for scoping method, pattern, and edge labeling.

## Overview

**Scope** is a foundational outcome type that lets you scope the entities created by an operation. You add it by right-clicking a **Step** and selecting **Insert > Scope**.

## Details View

### General

| Field | Description |
|---|---|
| **Outcome Type** | Displays the selected outcome type |

### Scope

| Field | Description |
|---|---|
| **Scoping Method** | Displays the scoping method for the outcome type |
| **Scoping Pattern** | Displays the scoping pattern for the outcome type |

### Definition

| Field | Description |
|---|---|
| **Label Edges** | Provides a label to identify the edges created during the operation |

> **Note:** In the **Merge Nodes** operation, **Label Edges** provides a label to identify the faces created after merging nodes.

## Steps That Produce This Outcome

- Any step where entity scoping is needed; added manually via **Insert > Scope**
