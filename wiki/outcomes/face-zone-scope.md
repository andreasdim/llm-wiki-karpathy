Outcome that allows scoping created face zones in an operation, making them available for subsequent operations.

## Overview

**Face Zone Scope** lets you scope the face zones created by an operation. Add it by right-clicking a **Step** and selecting **Insert > Face Zone Scope**.

## Details View

### General

| Field | Description |
|---|---|
| **Outcome Type** | Displays the selected outcome type |

### Scope

| Field | Description |
|---|---|
| **Scoping Method** | Displays the scoping method for the outcome type |
| **Scoping Pattern** | Displays the scoping pattern for the outcome type |

## Steps That Produce This Outcome

- Any step that creates face zones; added manually via **Insert > Face Zone Scope**
