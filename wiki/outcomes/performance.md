Beta outcome that displays the time taken and memory usage for a completed operation.

## Overview

**Performance** (Beta) lets you view the elapsed time and memory consumption of an operation. Add it by right-clicking a **Step** and selecting **Insert > Performance**.

## Details View

### General

| Field | Description |
|---|---|
| **Outcome Type** | Displays the selected outcome type |

### Scope

| Field | Description |
|---|---|
| **Elapsed Time** | Time taken to compute the operation |
| **Peak Process Memory** | Peak memory usage during the operation |
| **Final Process Memory** | Final memory usage after the operation |

## Steps That Produce This Outcome

- Any operation; added manually via **Insert > Performance**
