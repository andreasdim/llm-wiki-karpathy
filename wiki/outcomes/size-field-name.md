Outcome that allows scoping the name of a created size field, making it available for subsequent operations.

## Overview

**Size Field Name** lets you scope the name of a size field created by an operation. Add it by right-clicking a **Step** and selecting **Insert > Size Field Name**.

## Details View

### General

| Field | Description |
|---|---|
| **Outcome Type** | Displays the selected outcome type |

### Definition

| Field | Description |
|---|---|
| **Size Field Name Pattern** | Provides the name of the created size field and adds the name under the **Size Fields** tab in the **Domain Browser** |

## Steps That Produce This Outcome

- Steps that create size fields; added manually via **Insert > Size Field Name**
