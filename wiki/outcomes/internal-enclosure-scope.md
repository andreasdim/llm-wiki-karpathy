Outcome that allows scoping the internal enclosure created in an operation, making it available for subsequent operations.

## Overview

**Internal Enclosure Scope** lets you scope the internal enclosure created by an operation. Add it by right-clicking a **Step** and selecting **Insert > Internal Enclosure Scope**.

## Details View

### General

| Field | Description |
|---|---|
| **Outcome Type** | Displays the selected outcome type |

### Scope

| Field | Description |
|---|---|
| **Scoping Method** | Displays the scoping method for the outcome type |
| **Scoping Pattern** | Displays the scoping pattern for the outcome type |

## Steps That Produce This Outcome

- Steps that create internal enclosures; added manually via **Insert > Internal Enclosure Scope**
