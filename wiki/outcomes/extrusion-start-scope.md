Outcome that allows scoping the base face used for extruding, making it available for subsequent operations.

## Overview

**Extrusion Start Scope** lets you scope the base face used for an extrusion operation. Add it by right-clicking a **Step** and selecting **Insert > Extrusion Start Scope**.

## Steps That Produce This Outcome

- Extrusion operations; added manually via **Insert > Extrusion Start Scope**
