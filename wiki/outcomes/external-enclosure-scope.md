Outcome that allows scoping the external enclosure created in an operation, making it available for subsequent operations.

## Overview

**External Enclosure Scope** lets you scope the external enclosure created by an operation. Add it by right-clicking a **Step** and selecting **Insert > External Enclosure Scope**.

## Details View

### General

| Field | Description |
|---|---|
| **Outcome Type** | Displays the selected outcome type |

### Scope

| Field | Description |
|---|---|
| **Scoping Method** | Displays the scoping method for the outcome type |
| **Scoping Pattern** | Displays the scoping pattern for the outcome type |

## Steps That Produce This Outcome

- Steps that create external enclosures; added manually via **Insert > External Enclosure Scope**
