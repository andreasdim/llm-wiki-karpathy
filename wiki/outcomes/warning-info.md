Outcome that provides warning messages about issues that occurred in a mesh workflow step, automatically added upon completion.

## Overview

**Warning Info** reports warning messages related to issues in a mesh workflow operation. It is automatically added to the completed operation and cannot be added manually.

## Details View

### General

| Field | Description |
|---|---|
| **Outcome Type** | Displays the selected outcome type |

### Definition

| Field | Description |
|---|---|
| **Warning Text** | Provides the warning messages for the mesh workflow operation |

## Steps That Produce This Outcome

- Automatically added to any completed operation that generates warnings; cannot be inserted manually
