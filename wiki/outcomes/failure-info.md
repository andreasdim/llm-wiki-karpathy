Outcome that provides information about failures that occurred in a mesh workflow step, automatically added upon failure.

## Overview

**Failure Info** is an outcome that reports error details when a mesh workflow operation fails. It is automatically added to the operation upon failure and cannot be added manually.

## Details View

### General

| Field | Description |
|---|---|
| **Outcome Type** | Displays the selected outcome type |

### Definition

| Field | Description |
|---|---|
| **Error Text** | Provides the error messages for the failed mesh workflow operation |

## Steps That Produce This Outcome

- Automatically added to any operation that fails; cannot be inserted manually
