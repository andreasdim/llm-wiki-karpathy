Overview of the Property Worksheet for viewing and editing published properties of mesh workflow controls and step settings.

## Key Facts

- The **Property Worksheet** displays published properties of each control and the settings defined in the Steps Detail View.
- Provides easy visualization and inline editing of driving input options for mesh workflows.
- Also displays default published properties for the respective Mesh Workflow type.

## Columns and Options

- **Publish Properties**: Add unpublished properties for selected controls or steps. Select a checkbox and click Publish.
- **Show Association**: Toggle Association and Property columns.
- **Association**: Path of the step, control, or outcome using the published property.
- **Property**: The option published from the Details view or Publish Properties.
- **Description**: Function description; editable for custom text. Defaults to the path of the associated step/control/outcome.
- **Value**: Editable property values from the worksheet without accessing individual control types.

## Navigation

- **Go to page**: Navigate to a specified page.
- **Show rows**: Set the number of rows per page.
- Previous / Next page buttons.

## Right-Click Options

- **Unpublish**: Remove the selected row.
- **Go To Object**: View the selected control in the tree outline.
- **Define By Settings**: Set the property to be defined by settings (applies corresponding values from Steps Details view).
- **Define By Value**: Define the value directly from the worksheet.
