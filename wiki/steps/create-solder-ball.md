Creates solder ball geometry with specified parameters and mesh settings for electronics assembly simulation.

## Controls

- **Solder Ball Creation** — define shape (Sphere/Torus), dimensions, mesh divisions
- **Solder Ball User Names** — specify label names for created entities

## Outcomes

- **Solder Ball Top Pad**, **Solder Ball Bottom Pad**, **Scope**
