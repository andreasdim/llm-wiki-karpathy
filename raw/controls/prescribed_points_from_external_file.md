## Prescribed Points From External File

The **Prescribed Points From External File** control allows you to create the prescribed points based on the locations defined in a external .csv or .txt file.

The **Prescribed Points From External File** control **Details** view has the following options:

**General**

* **[Control Type](../controls.md)**

**Scope**

* **[Define By](../controls.md)**
* **[Scoping Method](../controls.md)**
* **[Scoping Pattern](../controls.md)**

Definition


* **File Name**: Allows you to import the node locations and node names from the .csv or .txt file.

* **Preserve Boundaries**: Allows you to preserve the nodes on boundaries when **Preserve Boundaries** set to **Yes**. 
  When Preserve Boundaries set to **No** the boundaries nodes moves to the nearest prescribed points.
