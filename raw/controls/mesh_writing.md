# Mesh Writing

**Mesh Writing** allows you to write and save the mesh generated 
in the mesh workflow for the selected step.
**Mesh Writing** supports .pmdat, .msh, .cdb file extension to write your mesh.

**Mesh Writing Details** view has the following options:

**General**

* **[Control Type](../controls.md)**

**Definition**

* **File Name**: Allows you to provide name and the path to write the file.
