#Leak Detection

**Leak Detection** allows you to visualize the leakages in the input scope so that you can close it.

**Leak Detection Details** view has the following options: 

**General** 

* **Control Type**: Displays the selected control type. 

**Definition** 

**Max Hole Size**: Allows you to provide the maximum hole size 
to prevent leakage. 
You can click  ![forward](../resources/publish.png)  on the 
right corner of the option and click **Publish** to add 
**Max Hole Size** to the **Property Worksheet**. 
You can parameterize **Max Hole Size**.
