# Topology Diagnostics User Names

**Topology Diagnostics User Names** control allows you to provide names to the labels 
created for the identified problematic topologies in the model. When you do not specify the name for the labels created for the diagnostics in the problematic topology, **Topology Diagnostics User Names** control creates a label name for each of the diagnostics. Also,**Topology Diagnostics** outcome provides the diagnostic information.

**Topology Diagnostics User Names** control has the following options:

**General** 

* **Control Type**: Displays the selected control type. 

**Definition** 

* **Incomplete Boundaries Label Name**: Allows you to provide a name for the created incomplete boundaries label.
 You can click ![forward](../resources/publish.png)  on the right corner 
  of the option and click **Publish** to publish **Incomplete Boundaries Label Name** to the **Property Worksheet**.

* **Missing Boundaries Label Name**: Allows you to provide a name for the created missing boundaries label.
You can click ![forward](../resources/publish.png)  on the right corner 
  of the option and click **Publish** to publish **Missing Boundaries Label Name** to the **Property Worksheet**.
  
* **Invalid Boundaries Label Name**: Allows you to provide a name for the created invalid boundaries label.
You can click ![forward](../resources/publish.png)  on the right corner 
  of the option and click **Publish** to publish **Invalid Boundaries Label Name** to the **Property Worksheet**.

* **Intersecting Pairs Label Name**: Allows you to provide a name for the created intersecting pairs label.
You can click ![forward](../resources/publish.png)  on the right corner 
  of the option and click **Publish** to publish **Intersecting Pairs Label Name** to the **Property Worksheet**.

* **Overlapping Faces Label Name**: Allows you to provide a name for the created overlapping faces label.
You can click ![forward](../resources/publish.png)  on the right corner 
  of the option and click **Publish** to publish **Overlapping Faces Label Name** to the **Property Worksheet**.

* **Thin Stripes Label Name**: Allows you to provide a name for the created thin stripes label.
You can click ![forward](../resources/publish.png)  on the right corner 
  of the option and click **Publish** to publish **Thin Stripes Label Name** to the **Property Worksheet**.

* **Small Edges Label Name**: Allows you to provide a name for the created small edges label.
You can click ![forward](../resources/publish.png)  on the right corner 
  of the option and click **Publish** to publish **Small Edges Label Name** to the **Property Worksheet**.

* **Unconnected Free Edges Label Name**: Allows you to provide a name for the created unconnected free edges label.
You can click ![forward](../resources/publish.png)  on the right corner 
  of the option and click **Publish** to publish **Unconnected Free Edges Label Name** to the **Property Worksheet**.
