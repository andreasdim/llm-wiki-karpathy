# Advanced Scope

**Advanced Scope** control provides advanced morphing scope options to automatically identify the morphable region.

**Advanced Scope** Details view has the following options:

**General**

* **Control Type**: Displays the type of morph control used.


**Definition**

* **Free Edges as Fixed**: Allows you to specify whether to consider free edges as fixed while morphing.
 The default value is **Yes**. 
When **Free Edges as Fixed** is **Yes**,  you cannot modify the free edges. That is, the nodes are frozen.

* **Surface Scope Type**: Allows you to define the type of morphable surface.
 The default value is **Rings**. The available options are:
    * **Rings**: Allows you to select the morphable scope as the number of rings of the connected neighbor faces.
    * **Distance**: Allows you to select the morphable scope within the specified distance from the control scope.

* **Number of Rings**: Allows you to specify the number of rings to be considered as morphable while morphing. 
The default value is **1**.
You can click  ![forward](../resources/publish.png)  on the right corner of the option and click **Publish** to add **Number of Rings** to the **Property Worksheet**. 
You can parameterize **Number of Rings**.

* **Distance**: Allows you to specify the distance from the control scope to be considered for the morphable scope while morphing.
The default value is **0.0 mm**.
 You can click  ![forward](../resources/publish.png)  on the right corner of the option and click **Publish** to add **Distance** to the **Property Worksheet**. 
 You can parameterize **Distance**.
