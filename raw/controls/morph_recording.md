# Morph Recording

**Morph Recording** controls allows you to create and store the morphed file.

**General**

* **Control Type**: Displays the type of morph control used.

**Definition**

* **File Name**: Allows you to save the morph file in the selected folder.
When you do not specify the **File Name**, the morph file is automatically 
saved in the Mechanical database.

* **Name**: Allows you to provide the name for the morphed field.
You can click  ![forward](../resources/publish.png) on the right corner of 
the option and click **Publish** to publish **Name** to the **Property Worksheet**.
You can parameterize **Distance**.
