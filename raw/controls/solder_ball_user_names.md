# Solder Ball User Names

**Solder Ball User Names** control allows you to specify the name for the created labels of solder ball.

**Solder Ball User Names Details** view has the following options:

**General**

* **[Control Type](../controls.md)**: Displays the selected control type.

**Definition**

* **Center Solder Label Name**: Allows you to provide a name for the center solder label created.
    You can click  ![forward](../resources/publish.png) on the right corner of 
    the option and click **Publish** to publish **Center Solder Label Name** to the **Property Worksheet**.

* **Top Solder Label Name**: Allows you to provide a name for the top solder label created.
    You can click  ![forward](../resources/publish.png) on the right corner of 
    the option and click **Publish** to publish **Top Solder Label Name** to the **Property Worksheet**.

* **Bottom Solder Label Name**: Allows you to provide a name for the bottom solder label created.
    You can click  ![forward](../resources/publish.png) on the right corner of 
    the option and click **Publish** to publish **Bottom Solder Label Name** to the **Property Worksheet**.

* **Top Solder Pad Label Name**: Allows you to provide a name for the top solder pad label created.
    You can click  ![forward](../resources/publish.png) on the right corner of 
    the option and click **Publish** to publish **Top Solder Pad Label Name** to the **Property Worksheet**.

* **Bottom Solder Pad Label Name**: Allows you to provide a name for the bottom solder pad label created.
    You can click  ![forward](../resources/publish.png) on the right corner of 
    the option and click **Publish** to publish **Bottom Solder Pad Label Name** to the **Property Worksheet**.

* **Top Solder Pad Face Label Name**: Allows you to provide a name for the top solder pad face label created.
    You can click  ![forward](../resources/publish.png) on the right corner of 
    the option and click **Publish** to publish **Top Solder Pad Face Label Name** to the **Property Worksheet**.

* **Bottom Solder Pad Face Label Name**: Allows you to provide a name for the bottom solder pad face label created.
    You can click  ![forward](../resources/publish.png) on the right corner of 
    the option and click **Publish** to publish **Bottom Solder Pad Face Label Name** to the **Property Worksheet**.
