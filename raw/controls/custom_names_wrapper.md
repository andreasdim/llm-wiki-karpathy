# Custom Names Wrapper

**Custom Names Wrapper** control creates entities with the names you provide.

**Custom Names Wrapper Details** view has the following options:

**General**

* **[Control Type](../controls.md)**

**Definition**

* **Part Name**: Allows you to provide name for the part that the wrapper creates enclosing the volume.

* **Volume Zone Name**: Allows you to provide name for the volume zone of the created part.

* **Face Zone Name**: Allows you to provide name for the face zone forming the boundary of the enclosed volume.

* **Edge Label Name**: Allows you to label the created feature edges with the provided names.
