### Stackable Bodies Detection User Names

**Stackable Bodies Detection User Names** allows you to set the name for the detected stackable bodies.

**Stackable Bodies Detection Details** view has the following options:

**General**

* **[Control Type](../controls.md)**

**Definition**

* **Name**: Allows you to set the name for the stackable bodies. You can click ![forward](../resources/publish.png)  on the right corner of the option and click **Publish** to publish **Name** to the **Property Worksheet**.
