# Surface Mesh Diagnostics User Name

**Surface Mesh Diagnostics User Name** control allows you to provide names to the labels 
created for self intersections, free mesh edges and multi mesh edges. When you have 
self intersections, free mesh edges or multi mesh edges in the created surface mesh,
 **Surface Mesh Diagnostics User Name** control creates a label name for each of the 
 diagnostics. When you do not specify the label name, **Surface Mesh Diagnostics User Name** 
 control creates labels with default names. Also, creates **Surface Mesh Diagnostic** 
 outcome that provides the number of self intersections, free mesh edges,
  multi mesh edges created during the surface mesh operation.

**Surface Mesh Diagnostics User Name** Details view has the following options:


**General**

* **[Control Type](../controls.md)**: Displays the selected control type.

**Definition**

* **Self Intersections Label Name**: Allows you to provide a name for the self intersection label created.
You can click  ![forward](../resources/publish.png) on the right corner of 
the option and click **Publish** to publish **Self Intersections Label Name** to the **Property Worksheet**.
* **Free Mesh Edges Label Name**: Allows you to provide a name for the free mesh edges label created.
You can click  ![forward](../resources/publish.png) on the right corner of 
the option and click **Publish** to publish **Free Mesh Edges Label Name** to the **Property Worksheet**.
* **Multi Mesh Edges Label Name**: Allows you to provide a name for the multi mesh edges label created.
You can click  ![forward](../resources/publish.png) on the right corner of 
the option and click **Publish** to publish  **Multi Mesh Edges Label Name** to the **Property Worksheet**.

* **Volumes Label Name(Beta)**:  Allows you to provide a name for the number of volumes scoped label created.
You can click  ![forward](../resources/publish.png) on the right corner of 
the option and click **Publish** to publish  **Volumes Label Name** to the **Property Worksheet**.

* **Faces with Skewed Elements Label Name(Beta)**: Allows you to provide a name for the faces with skewed elements label created.
You can click  ![forward](../resources/publish.png) on the right corner of 
the option and click **Publish** to publish **Faces with Skewed Elements Label Name** to the **Property Worksheet**.
