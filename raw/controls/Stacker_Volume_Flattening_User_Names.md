### Stacker: Volume Flattening User Names

**Stacker: Volume Flattening User Names** allows you to set the name for the base face created in **Stacker: Volume Flattening** control.

**Stacker: Volume Flattening User Names** view has the following options:

**General**

* **[Control Type](../controls.md)**: Displays the selected control type.


**Definition**

* **Base Face Label**: Provides the name for the base face created in **Stacker: Volume Flattening** control. You can click ![forward](../resources/publish.png)  on the right corner of the option and click **Publish** to publish **Base Face Label** to the **Property Worksheet**.
