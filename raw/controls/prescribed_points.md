## Prescribed Points 

The **Prescribed Points** control allows you to Creates the prescribed points based on the locations of the defined material points.
The **Prescribed Points** control  **Details** view has the following options:

**General**

* **[Control Type](../controls.md)**

**Scope**

* **[Define By](../controls.md)**
* **[Scoping Method](../controls.md)**
* **[Scoping Pattern](../controls.md)**

**Definition**

* **Label Name**: Allows you to provide the label name for the prescribed point.
* **Unique Label Per Point**: Automatically creates the unique label name for defined points by adding a numeric suffix to the **Label Name**, when **Unique Label Per Point** set to **Yes**.
* **Preserve Boundaries**: Allows you to preserve the nodes on boundaries when **Preserve Boundaries** set to **Yes**.
  When Preserve Boundaries set to **No** the boundaries nodes moves to the nearest prescribed points.
