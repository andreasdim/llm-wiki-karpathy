# Mesh Reading

**Mesh Reading** allows you to read a mesh into the mesh workflow model.
The read mesh can be either appended to the already existing mesh or overwrite it. 
**Mesh Reading** supports .pmdat, .msh, .cdb file extension for reading the mesh.

**Mesh Reading Details** view has the following options:

**General**

* **[Control Type](../controls.md)**

**Definition**
* **File Name**: Allows you to provide path of the file to be read.

* **Append Data**: Allows you to append the mesh data from the 
file to the mesh workflow model.

* **Repair Model**: Allows you to repair the mesh workflow model after reading 
the mesh data, creating missing zones and volumes if required.
