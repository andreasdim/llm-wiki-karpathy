### MultiZone Surface Diagnostics

**MultiZone Surface Diagnostics** outcome provides information about the failures and defeatures 
that occurred while performing multizone surface meshing.

**MultiZone Surface Diagnostics** Details view has the following options:

**General**

* **[Outcome Type](../outcomes/scope.md)**


**Scope**

* **Failed Blocking Scoping Method**: Displays the scoping method used for scoping failed blocking.
* **Failed Blocking Scoping Pattern**: Displays the scoping pattern used to scope failed blocking.
* **Free Mesh Scoping Method**: Displays the scoping method used for scoping free mesh.
* **Free Mesh Scoping Pattern**: Displays the scoping pattern used to scope free mesh.
* **Failed Meshing Scoping Method**: Displays the scoping method used for scoping failed meshing.
* **Failed Meshing Scoping Pattern**: Displays the scoping pattern used to scope failed meshing.
* **Failed Input Mesh Connection Scoping Method**: Displays the scoping method used for 
    scoping failed input mesh connection.
* **Failed Input Mesh Connection Scoping Pattern**: Displays the scoping pattern used to 
    scope failed input mesh connection.
* **Failed Mesh Check Scoping Method**: Displays the scoping method used for scoping failed mesh check.
* **Failed Mesh Check Scoping Pattern**: Displays the scoping pattern used to scope failed mesh check.
* **Defeature Edges Scoping Method**: Displays the scoping method used for scoping defeature edges.
* **Defeature Edges Scoping Pattern**: Displays the scoping pattern used to scope defeature edges.
* **Defeature Faces Scoping Method**: Displays the scoping method used for scoping defeature faces.
* **Defeature Faces Scoping Pattern**: Displays the scoping pattern used to scope defeature faces.

**Definition**

* **Failed Blocking**: Displays the number of entities where the MultiZone blocking fails.
* **Free Mesh**: Displays the number of entities where the MultiZone blocking failed 
    to create mapped or swept mesh.
* **Failed Meshing**: Displays the number of entities where the MultiZone failed to create mesh.
* **Failed Input Mesh Connection**: Displays the number of entities where the MultiZone 
    failed to conformally connect to the input mesh.
* **Failed Mesh Check**: Displays the number of entities where the MultiZone created
    inverted or invalid elements.
* **Defeature Edges**: Displays the number of defeatured edges.
* **Defeature Faces**: Displays the number of defeatured faces.
