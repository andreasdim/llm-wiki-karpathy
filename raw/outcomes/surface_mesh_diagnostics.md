# Surface Mesh Diagnostics

**Surface Mesh Diagnostics** outcome provides the number of self intersections, free mesh edges and multi mesh edges created on the surface mesh.

>Note: When the **Mesh Surface** operation fails, the control automatically adds the **Surface Mesh Diagnostics** outcomes.

**Surface Mesh Diagnostics** Details view has the following options:

**General**

* **Outcome Type**: Displays the selected outcome type.

**Scope**

* **Self Intersections Scoping Method**: Displays the scoping method used for scoping self intersections
* **Self Intersections Scoping Pattern**: Displays the scoping pattern used to scope self intersections.
* **Free Mesh Edges Scoping Method**: Displays the scoping method used for scoping free mesh edges.
* **Free Mesh Edges Scoping Pattern**: Displays the scoping pattern used to scope free mesh edges.
* **Multi Mesh Edges Scoping Method**: Displays the scoping method used for scoping multi mesh edges.
* **Multi Mesh Edges Scoping Pattern**: Displays the scoping pattern used to scope multi mesh edges.
* **Volumes Scoping Method**: Displays the scoping method used for scoping volumes.
* **Volumes Scoping Pattern**: Displays the scoping pattern used to scope volumes.
* **Faces with Skewed Elements Scoping Method**: Displays the scoping method 
    used for scoping faces with skewed elements.
* **Faces with Skewed Elements Scoping Pattern**: Displays the scoping pattern used 
    to scope faces with skewed elements.

**Definition**

* **Number of Self Intersections**: Provides the number of self intersection in the created surface mesh.

* **Number of Free Mesh Edges**: Provides the number of free mesh edges in the created surface mesh.

* **Number of Multi Mesh Edges**: Provides the number of multi-mesh edges in the created surface mesh.

* **Number of Volumes**: Provides the number of volumes in the created surface mesh.

* **Number of Faces with Skewed Elements**: Provides the number of faces with skewed elements in the created surface mesh.
