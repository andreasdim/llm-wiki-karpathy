# Solder Ball Top Pad

**Solder Ball Top Pad** outcome allows you to scope the created solder ball bottom pad of the operation.


**Solder Ball Top Pad Details** view has the following options:

**General**

* **Outcome Type**: Displays the selected outcome type.

**Scope**

* **Solder Ball Top Pad Scoping Method**: Displays the scoping method for the **Solder Ball Top Pad**.
* **Solder Ball Top Pad Scoping Pattern**: Displays the scoping pattern for the **Solder Ball Top Pad**.
