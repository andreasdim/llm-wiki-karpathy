## Warning Info

**Warning Info** provides information about the warnings related to the issues that occurred in the respective mesh workflow step.

>**Note**:  **Warning Info Outcome** is automatically added to the completed operation. You cannot add **Warning Info** as **Outcome** manually.

**Warning Info Details** view has the following options:

**General**

* **[Outcome Type](../outcomes/scope.md)**

**Definition**

* **Warning Text**: Provides you the warning messages for the mesh workflow operation.
