# Failure Info

**Failure Info** provides information about the failures that occurred in the respective mesh workflow step.

>**Note**:  **Failure Info Outcome** is automatically added to its operation upon failure. You cannot add **Failure Info** as **Outcome** manually.

**Failure Info Details** view has the following options:

**General**

* **[Outcome Type](../outcomes/scope.md)**

**Definition**

* **Error Text**: Provides you the error messages for the failed mesh workflow operation.
