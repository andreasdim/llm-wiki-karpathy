**Stacker Base Edge Scope** outcome allows you to scope the created base face edge of the operation.


**Stacker Base Edge Scope** view has the following options:

**General**

* **Outcome Type**: Displays the selected outcome type.

**Scope**

* **Scoping Method**: Displays the scoping method for the **Stacker Base Edge Scope**.
* **Scoping Pattern**: Displays the scoping pattern for the **Stacker Base Edge Scope**.
