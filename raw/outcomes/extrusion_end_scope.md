# Extrusion End Scope
**Extrusion End Scope** allows you to scope the end face created after extrusion.\
Right-click the **Step** and click **Insert > Extrusion End Scope** to add extrusion end as an outcome and make it available for scoping in the subsequent operations. 

**Extrusion End Scope Details** view has the following options:

**General**
* **[Outcome Type](../outcomes/scope.md)**

**Scope**

* **[Scoping Method](../outcomes/scope.md)**
* **[Scoping Pattern](../outcomes/scope.md)**
