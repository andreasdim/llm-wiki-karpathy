# Extrusion Start Scope
**Extrusion Start Scope** allows you to scope the base face used for extruding.\
Right-click the **Step** and click **Insert > Extrusion Start Scope** to add extrusion start scope as an outcome and make it available for scoping in the subsequent operations.

**Extrusion Start Scope Details** view has the following options:

**General**
* **[Outcome Type](../outcomes/scope.md)**

**Scope**

* **[Scoping Method](../outcomes/scope.md)**
* **[Scoping Pattern](../outcomes/scope.md)**
