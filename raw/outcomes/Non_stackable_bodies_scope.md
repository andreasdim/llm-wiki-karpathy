## Non-Stackable Bodies Scope
**Non-Stackable Bodies Scope** outcome allows you to scope the created non-stackable bodies of the operation.


**Non-Stackable Bodies Scope Details** view has the following options:

**General**

* **Outcome Type**: Displays the selected outcome type.

**Scope**

* **Scoping Method**: Displays the scoping method for the outcome type.
* **Scoping Pattern**: Displays the scoping pattern for the outcome type.
