# Scope

**Scope** outcome allows you to scope the created entities of the operation.
Right-click the **Step** and click **Insert > Scope** to add **Scope** as an outcome.

**Scope Details** view has the following options:

**General**

* **Outcome Type**: Displays the selected outcome type.

**Scope**

* **Scoping Method**: Displays the scoping method for the outcome type.
* **Scoping Pattern**: Displays the scoping pattern for the outcome type.

**Definition**

* **Label Edges**: Provides label to identify the edges created during the operation.

>Note: In **Merge Nodes** operation, **Label Edges** provides label to identify the faces created after merging nodes.
