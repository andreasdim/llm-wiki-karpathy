# Solder Ball Bottom Pad

**Solder Ball Bottom Pad** outcome allows you to scope the created solder ball bottom pad of the operation.


**Solder Ball Bottom Pad Details** view has the following options:

**General**

* **Outcome Type**: Displays the selected outcome type.

**Scope**

* **Solder Ball Bottom Pad Scoping Method**: Displays the scoping method for the **Solder Ball Bottom Pad**.
* **Solder Ball Bottom Pad Scoping Pattern**: Displays the scoping pattern for the **Solder Ball Bottom Pad**.
