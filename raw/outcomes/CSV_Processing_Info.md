## CSV Processing Info

**CSV Processing Info** provides information about the failure that occurred in the .CSV file for the respective mesh workflow step.

>**Note**:  **CSV Processing Info** is automatically added to the completed operation. You cannot add **CSV Processing Info** as **Outcome** manually.

**CSV Processing Info** view has the following options:

**General**

* **[Outcome Type](../outcomes/scope.md)**

**Definition**

* **Lines Skipped**: Displays the skipped line in the respective .csv file.
* **Failure Lines**: Displays the failed line numbers in the respective .csv file that caused the operation to fail or provide warning.
