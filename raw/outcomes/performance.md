# Performance(Beta)

**Performance** allows you to view the time taken to complete the selected operation
and its memory usage.
Right-click the **Step**, click **Insert > Performance** to add performance as an outcome and make it available for scoping in the subsequent operations.

**Performance Details** view has the following options:

**General**
* **[Outcome Type](../outcomes/scope.md)**

**Scope**

* **Elapsed Time**: Displays the time taken to compute the operation.
* **Peak Process Memory**: Displays the peak memory usage.
* **Final Process Memory**: Displays the final memory usage.
