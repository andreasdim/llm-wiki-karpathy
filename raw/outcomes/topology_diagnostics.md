# Topology Diagnostics

**Topology Diagnostics** outcome provides the diagnostic information about the topology. The information includes the number of missing boundaries, incomplete boundaries and invalid boundaries in the topology.

>Note: When the **Mesh Surface** operation fails, the control automatically adds the **Topology Diagnostics** outcomes.

Right-click the **Step** and click **Insert > Topology Diagnostics** to add topology diagnostics as an outcome and make it available for scoping in the subsequent operations.

**Topology Diagnostics** Details view has the following options:

**General**

* **[Outcome Type](../outcomes/scope.md)**:  Displays the selected outcome type.

**Scope**

* **Incomplete Boundaries Scoping Method**: Displays the scoping method used for scoping incomplete boundaries.
* **Incomplete Boundaries Scoping Pattern**: Displays the scoping pattern used to scope incomplete boundaries.
* **Missing Boundaries Scoping Method**: Displays the scoping method used for scoping missing boundaries.
* **Missing Boundaries Scoping Pattern**: Displays the scoping pattern used to scope missing boundaries.
* **Invalid Boundaries Scoping Method**: Displays the scoping method used for scoping invalid boundaries.
* **Invalid Boundaries Scoping Pattern**: Displays the scoping pattern used to scope invalid boundaries.
* **Unconnected Free Edges Scoping Method**: Displays the scoping method used for unconnected free edges.
* **Unconnected Free Edges Scoping Pattern**:Displays the scoping pattern used to scope unconnected free edges.
* **Faces with Intersecting Pairs Scoping Method**: Displays the scoping method for scoping faces with intersecting pairs.
* **Faces with Intersecting Pairs Scoping Pattern**: Displays the scoping pattern used  to scope faces with intersecting pairs.
* **Overlapping Faces Scoping Method**: Displays the scoping method used for scoping overlapping faces.
* **Overlapping Faces Scoping Pattern**: Displays the scoping pattern used to scope overlapping faces.
* **Thin Stripes Scoping Method**: Displays the scoping method used for scoping thins stripes.
* **Thin Stripes Scoping Pattern**: Displays the scoping pattern used to scope thin stripes.
* **Small Edges Scoping Method**: Displays the scoping method used for scoping small edges.
* **Small Edges Scoping Pattern**: Displays the scoping pattern used to scope the small edges.

>Note: **Scope** is available only when there are diagnostics issues and displays the scoping method and created labels for each of the corresponding diagnostic issues.

**Definition**

* **Number of Incomplete Boundaries**: Provides the number of incomplete boundaries in the topology.
* **Number of Missing Boundaries**: Provides the number of missing boundaries in the topology.
* **Number of Invalid Boundaries**: Provides the number of invalid boundaries in the topology.
* **Number of Unconnected Free Edges**: Provides the number of unconnected free edges in the topology.
* **Number of Faces with Intersecting Pairs**: Provides the number of faces having intersecting pairs in the topology.
* **Number of Overlapping Faces**: Provides the number of overlapping faces in the topology.
* **Number of Thin Stripes**: Provides the number of thin stripes in the topology.
* **Number of Small Edges**: Provides the number of small edges in the topology.
