### Stacker Base Face Scope

**Stacker Base Face Scope** outcome allows you to scope the created base face of the operation.


**Stacker Base Face Scope** view has the following options:

**General**

* **Outcome Type**: Displays the selected outcome type.

**Scope**

* **Scoping Method**: Displays the scoping method for the **Stacker Base Face Scope**.
* **Scoping Pattern**: Displays the scoping pattern for the **Stacker Base Face Scope**.
