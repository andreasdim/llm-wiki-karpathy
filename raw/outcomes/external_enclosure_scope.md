# External Enclosure Scope
**External Enclosure Scope** allows you to scope the external enclosure created in the selected operation.\
Right-click the **Step**, click **Insert > External Enclosure Scope** to add external enclosure as an outcome and make it available for scoping in the subsequent operations.

**External Enclosure Scope Details** view has the following options:

**General**
* **[Outcome Type](../outcomes/scope.md)**

**Scope**

* **[Scoping Method](../outcomes/scope.md)**
* **[Scoping Pattern](../outcomes/scope.md)**
