### Auto Repair Topology

The **Auto Repair Topology** operation automatically improves topology using predefined order of repair options and helps to generate a high-quality mesh.


The **Automatic Repair Topology** operation has the following controls:

* [**Auto Repair**](../controls/Auto_Repair.md): Allows you to improve topology using predefined order of repair options on the defined scoped entities.

* [Topology Protection](../controls/topology_protection.md): Instructs the mesher algorithm to retain entities in the selected scope without defeaturing.
