# Merge Volumes

The **Merge Volumes** operation allows you to merge the volumes. 

The **Merge Volumes**operation has the following control:


* **[Volumes Merging](../controls/volumes_merging.md)**:  Merges all the volumes that share the input faces defined on the scope into a single volume.

The **Merge Volumes** operation has specific outcomes which can be added as per your needs.
You can right-click **Merge Volumes**, click **Insert** and select the required 
scope that you need as outcome in the consecutive operations.
The available outcome is:

* **[Volume Zone Scope](../outcomes/volume_zone_scope.md)**: Allows you to scope the created volume zones in the selected operation.
