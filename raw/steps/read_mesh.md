# Read Mesh
**Read Mesh** allows you to read or append the mesh from a supported file (.pmdat,.msh,.cdb) to the mesh workflow model.

> **Note**: You can only insert **Read Mesh** before the first Step in a predefined Mesh Workflow.

The **Operation Type** is **Read Mesh**.

 **Read Mesh** has the following controls:

* **[Mesh Reading](../controls/mesh_reading.md)**
