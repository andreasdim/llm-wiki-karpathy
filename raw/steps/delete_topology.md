# Delete Topology

The **Operation Type** is **Delete Topology**. **Delete Topology** allows you to delete the topology and retains only the mesh data of the model. 

**Delete Topology** has the following controls:

* **[Topology Deletion](../controls/topology_deletion.md)**


**Delete Topology** has specific outcomes which can be added as per your needs. You can right-click **Delete Topology**, click **Insert** and select the required scope that you need as outcome in the consecutive operations. The available outcome is:

* **[Performance](../outcomes/performance.md)**
