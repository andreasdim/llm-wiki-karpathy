# Improve Surface Mesh 


**Improve Surface Mesh** allows you to improve the surface mesh tessellation. 
The **Operation Type** is Improve Surface Mesh.

**Improve Surface Mesh** has the following controls:

* **[Second Order Conversion](../controls/second_order_conversion.md)**

* **[Quadrilaterals to Triangles Splitting](../controls/quadrilaterals_to_triangles_splitting.md)**
* **[Resolve Self Intersections](../controls/resolve_self_intersections.md)**
