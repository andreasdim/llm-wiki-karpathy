#### Replicate Mesh


 The **Replicate Mesh** operation duplicates the volume mesh pattern onto another location in the model. You can use **Replicate Mesh** operation to create identical mesh structures.



The **Replicate Mesh** operation  has the following controls:

[**Mesh Replication**](../controls/mesh_replication.md): Allows you to duplicate volume meshed geometry and replicate it to specified locations defined in the .csv file or .txt file.

The **Replicate Mesh** operation  has the following outcomes:

* [**Warning Info**](../outcomes/warning_info.md): Provides information about the warnings that occurred in the **Replicate Mesh** operation.

* [**CSV Processing Info**](../outcomes/CSV_Processing_Info.md): Provides the processing information about the .csv files that occurred in the **Replicate Mesh** operation.
