# Merge Nodes

**Merge Nodes** operation allows you to merge the source nodes to target nodes within the specified  merge node tolerance.

The **Merge Nodes** operation has the following control:

* **[Merge Nodes on Faces](../controls/merge_nodes_on_faces.md)**: Merges the source face nodes to the target face nodes within the provided merge node tolerance.


The **Merge Nodes** operation has specific outcomes which can be added as per your needs.
You can right-click **Merge Nodes**, click **Insert** and select the required 
scope that you need as outcome in the consecutive operations.
The available outcomes are:

* **[Scope](../outcomes/scope.md)**: Allows you to scope the newly created faces after the merge node operation.
* **[Merge Nodes Outcome](../outcomes/merge_nodes_outcome)**: Provides the number of nodes merged during the merge node operation.
