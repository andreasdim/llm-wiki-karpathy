### Stacker Diagnostics

The **Stacker: Diagnostics** operation diagnoses the scoped entities and finds the smallest lateral edge length, smallest stacker edge length and bounding box dimensions.

> Note: The **Stacker: Diagnostics** step is optional. You can delete it if **Stacker: Diagnostics** is not needed.

The **Stacker: Diagnostics** operation has the following control:
 * **[Stacker: Diagnostics](../controls/Stacker_Diagnostics.md)**: Diagnoses the scoped entities along the defined axis.

The **Stacker: Diagnostics**  operation has the following outcomes:
* **[Scope](../outcomes/scope.md)**: Scopes the entities for the **Stacker: Diagnostics** operation.
* **[Stacker: Diagnostics Outcome](../outcomes/Stacker_Diagnostics_Outcome.md)**: Displays the smallest lateral edge length, smallest stacker edge length and bounding box dimensions.
