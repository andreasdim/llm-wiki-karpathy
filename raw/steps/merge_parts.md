# Merge Parts

**Merge Parts** operation allows you to merge all parts defined in the scope to a single part. 
**Merge Parts** operation should be added before specific steps if you want to achieve shared topology or conformal mesh between bodies that reside in different parts.

**Merge Parts** operation has the following control:


* **[Parts Merging](../controls/parts_merging.md)**: Merges all the parts defined in the scope to a single part.


**Merge Parts** operation has specific outcomes which can be added as per your needs.
You can right-click **Merge Parts**, click **Insert** and select the required 
scope that you need as outcome in the consecutive operations.
The available outcome is:

* **[Scope](../outcomes/scope.md)**: Allows you to scope the created entities in the selected operation.
