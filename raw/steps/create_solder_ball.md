# Create Solder Ball

The **Create Solder Ball**  operation allows you to create a solder ball using defined solder ball parameters and mesh settings. 

The **Create Solder Ball** operation has the following controls:

* **[Solder Ball Creation](../controls/solder_ball_creation.md)**: Define the solder ball parameters and mesh settings for the solder ball creation.
* **[Solder Ball User Names](../controls/solder_ball_user_names.md)**: Specifies the name for the created labels of the solder ball.

The **Create Solder Ball** operation has the following Outcomes:

* **[Solder Ball Top Pad](../outcomes/solder_ball_top_pad.md)**: Scopes the created solder ball top pad of the operation.
* **[Solder Ball Bottom Pad](../outcomes/solder_ball_bottom_pad.md)**: Scopes the created solder ball bottom pad of the operation.

* **[Scope](../outcomes/scope.md)**: Scopes the created labels of the **Create Solder Ball** operation.
