### Stacker: Detect Stackable Bodies

The **Stacker: Detect Stackable Bodies** operation detects the stackable bodies along stacking direction defined in the **Stacker: Stackable Bodies Detection** control and create the labels for the stackable and non stackable bodies.

The **Stacker: Detect Stackable Bodies** operation has the following controls:

* **[Stackable Bodies Detection](../controls/stackable_bodies_detection.md)**: Detects the stackable bodies along the defined axis.
* **[Stackable Bodies Detection User Names](../controls/Stackable_Bodies_Detection_User_Names.md)**: Allows you to set the name for the detected stackable bodies.

The **Stacker: Detect Stackable Bodies** operation has the following outcomes:
* **[Scope](../outcomes/scope.md)**: Scopes the created labels for the stackable bodies of the **Stacker: Detect Stackable Bodies** operation.
* **[Non Stackable Bodies Scope](../outcomes/Non_stackable_bodies_scope.md)**: Scopes the created labels for the non stackable bodies of the **Stacker: Detect Stackable Bodies** operation.
