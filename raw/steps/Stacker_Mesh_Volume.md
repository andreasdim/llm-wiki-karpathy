### Stacker: Mesh Volume

The ***Stacker: Mesh Volume** operation creates volume mesh from the base face with number of divisions along the defined edges.

The **Stacker: Mesh Volume** operation has the following controls:

* [**Stacker: Volume Meshing**](../controls/stacker_volume_meshing.md): Creates the volumetric mesh by extruding the meshed 2D base face.
* [**Number of Divisions on Edges**](../controls/Number_of_Division_on_Edges.md): Defines the number of divisions along the scoped edge labels along Stacking Direction.


The **Stacker: Mesh Volume** operation has the following outcomes:
* **[Scope](../outcomes/scope.md)**: Scopes the created labels for the stackable bodies of the **Stacker: Mesh Volume** operation.
