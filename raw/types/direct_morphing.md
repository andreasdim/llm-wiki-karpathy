# Direct Morphing

**Direct Morphing** workflow helps you transfer a mesh created directly 
in Mechanical, apply a morphing on the scoped region and send the final 
morphed mesh to mechanical. The Direct Morphing workflow can create a 
file with the prescribed displacements using the Morph Recording 
control and outcome.  You can apply the created morph files to the **Mesh** 
domain directly through **Morph Control** in **Mesh Edit**. 

When you select **Mesh Workflows** as **Direct Morphing**, 
**Mesh Workflow** loads a predefined template with **Steps** and **Outcomes**.
**Mesh Workflow** performs these **Steps** through **Controls** and
**Outcomes** to achieve the desired mesh for the **Direct Morphing**.

**Direct Morphing** workflow has the following step:

- [Direct Morph](../steps/direct_morph.md)
